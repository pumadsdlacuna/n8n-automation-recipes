# Example 01 – API Integration Automation

## EN
This recipe demonstrates how to integrate an external API with validation and retries.

## ES
Esta receta muestra cómo integrar una API externa con validación y reintentos.

---

## Workflow Steps

1. Trigger (Webhook or Schedule)
2. Validate input
3. Call external API
4. Handle errors
5. Store or forward results
